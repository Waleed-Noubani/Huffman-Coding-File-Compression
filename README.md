# Huffman Coding – File Compression

This project implements a lossless file compression and decompression system using the Huffman Coding algorithm.

## Overview
- Count byte frequencies from input file
- Build Huffman tree using a Priority Queue
- Generate prefix codes
- Compress file and write encoded output
- Decompress file and verify it matches the original

## Requirements
- Custom Binary Tree
- Custom Priority Queue
- GUI Interface
- Follow lecture implementation steps
