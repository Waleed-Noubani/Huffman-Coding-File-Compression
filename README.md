# Huffman Coding – File Compression System

## Overview
A lossless file compression and decompression system implemented using the Huffman Coding algorithm.  
The system reads any input file, builds a Huffman Tree based on byte frequencies, generates prefix codes, and produces a compressed file that can be accurately decompressed back to the original content.

## Technical Implementation
- Frequency analysis of file bytes
- Huffman Tree construction using a Priority Queue (Min-Heap)
- Prefix code generation
- Bit-level file encoding and decoding
- Header creation for storing encoding metadata
- GUI-based user interface

## Features
- Lossless compression
- Automatic decompression verification
- Custom Binary Tree implementation
- Custom Priority Queue implementation

## Complexity
Time Complexity: O(n log n)
